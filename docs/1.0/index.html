    py2ren
    • Welcome to py2ren’s documentation!

    Welcome to py2ren’s documentation!

    py2ren is a tool that will allow you to transform python module into ren’py store module.

    This transformation is done by taking the code and copying it to .rpy files wrapped inside the expressions for a module stored in ren’py like init python in module_name
