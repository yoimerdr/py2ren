    cli

    The cli module provides command-line interface functionality: argument parsing and execution of the conversion process through the main entry point.

    class Args(path: str, out: str = None, config: str | ModuleConfig | FileConfig = None, init_level: int = None, name: str = None, no_keep_structure: bool = False, force_config: bool = False)

    The cli args class.

    Parameters:

    • path – The target path.

    • out – The output folder.

    • config – The config path or instance.

    • init_level – The init python level.

    • name – The name of the target as store module.

    • no_keep_structure – If true, no keeps the structure of the source folder.

    • force_config – If true, creates and dumps a new configuration.
