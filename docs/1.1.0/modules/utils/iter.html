    iter

    The iter module provides iteration utility functions with Python 2/3 compatibility, and other helper functions for iterable operations.

    itermap(fn: Callable[[_T], _R], iterable: Iterable[_T]) → Generator[_R]

    Map a function over an iterable lazily.

    Compatibility function for python 2. In python 3.x, its equivalent to map().

    Parameters:

    • fn – Function to apply to each element

    • iterable – Input iterable

    iterfilter(fn: Callable[[_T], bool], iterable: Iterable[_T]) → Generator[_T]

    Filter an iterable lazily based on a predicate.

    Compatibility function for python 2. In python 3.x, its equivalent to filter().

    Parameters:

    • fn – Predicate function returning true for elements to keep

    • iterable – Input iterable

    iterrange(start: int = None, stop: int = None, step: int = None) → Generator[int]

    Create a lazy range generator with optional start, stop and step parameters.

    Compatibility function. In python 2 is equivalent to xrange(); and in 3, its equivalent to range().

    Parameters:

    • start – Starting value (inclusive).

    • stop – Ending value (exclusive).

    • step – Step size between values.

    first(iterable: Iterable[_T], default: _T = None, key: Callable[[_T], bool] = None) → _T

    Get the first element from an iterable that matches an optional key function.

    Parameters:

    • iterable – Input iterable

    • default – Value to return if no element is found

    • key – Predicate function to filter the first element.

    Returns:

    First matching element or default value

    last(iterable: Iterable[_T], default: _T = None, key: Callable[[_T], bool] = None)

    Get the last element from an iterable that matches an optional key function.

    Parameters:

    • iterable – Input iterable

    • default – Value to return if no element is found

    • key – Predicate function to filter the last element.

    Returns:

    Last matching element or default value
